### Pre-requisites

* NodeJS (v10.6.0 used )
* NPM (6.1.0 used)

### Setup
  Clone the repository to your local machine.
  ```
    npm install
    npm run start:prod  //Open http://localhost:3000
    npm run start:dev   //Open http://localhost:8080
    npm run test       //to run test cases
  ```
